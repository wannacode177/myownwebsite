# 写在新年之前

A little step for this blog,a huge step for me.

# 引言

首先，我想感谢自己没有在实现建博客梦想的过程中放弃。也正应了那句话，“世上无难事，只要肯攀登”。我决定把这句话当作自己的座右铭。博客顺利上线时本想着终于可以休息了，但又念及一份好的 “开博词” 往往能为其博客奠定未来的基调，于是又打开电脑来写这样一篇博文。倒也无妨，那我就从这一整个艰难的建博过程说起吧。

# 曲折的探索

其实在新学期刚开始的时候，我就已经有了自建博客的想法了。
我平时极少甚至可以说是从不在 qq、微信这样的社交平台上发声，因此许多朋友都觉得我是个沉稳不爱说话的人，其实我只是与大多数身边人鲜有共同话题罢了。就比如朋友圈里大都是诸如探店打卡、吃喝玩乐之类，而我想看到的却是一些文章的读后感抑或一些技术内容分享什么的。。。。。。貌似我在慢慢向宅男、极客这类发展。谁不想当现充啊喂！
这样一想，在某天午后突然萌生 “自己建一个这样的平台，吸引志同道合的伙伴交流” 的想法也是自然而然了。无奈学业繁重，兼之新学期新环境给自己立了太多 flag（也没完成多少），所以把这件事拖到了寒假。插一句，大学的寒假，真香！感觉这时候才能体会到人生中少有的、宝贵的自由。于是寒假过半，才真正开始着手建博客。

# 前期准备

放假前从图书馆借得一本叫做《跟我一起做网站》的 13 年老书，看了看目录发觉域名、虚拟主机等尚可实现，但网站的设计、架构对于我这样一个小菜鸡未免过于困难。遂合上书转投搜索引擎的怀抱。最初，结合网上推荐我打算利用 wordpress 和学生价云服务器来建站，腾讯云的学生价一年只有一百多点，还没到原价的十分之一。针不辍！刚准备付款的时候大数据推荐终于起了点作用，知乎给我推送了一篇关于用 Hexo 和 github 零成本 (如果我在实操过程中损失的大量时间成本不算成本的话）零基础建博客的教程。这不正适合我这种人（哪种？）嘛？二话不说，开干！

# 着手建博

这里也是希望后来者能少走我走的弯路啊 qwq

按照教程指导，首先要下载 git 以及 node.js ，然后再创建 ssh 密钥并关联到 github 上来搭建 hexo 的必须环境 —— 这里由于我是 win 系统上装有 linux 虚拟机，而且虚拟机已经安装好了那两个软件，所以选择到 linux 系统里继续接下来的操作 —— 又装好一系列东东之后，测试一下已经可以在本地浏览初始样式的博客了，我那叫一个开心啊！就差最后一步把本地博客推送到 github 上就大功告成了！然！而！就是这最后一步折磨了我两个下午。推送一直无法顺利进行。这时候又是知乎前来助我：

知乎：现在知道我的好了吧？

正巧有人在同一时间问了同样的问题，吸引来了大佬回答：“错的不是你，是全世界！” 其实就是 hexo-deployer-git 这个应用程序临时出了问题，删了重下个别的就成了。以下是代码，亲测有效，贴上来也是防止以后再遇到这种情况。

npm un hexo-deployer-git

npm i hexojs/hexo-deployer-git

这样在重新操作之后再输入网址，哇，好耶，网站正式上线 1.0 版本！不得不说那一刻还是很开心的，心中满满的成就感。

# 博客美化

人总是不容易满足。我觉得初始主题肯定不够好，于是去主题站选主题，千挑万选选中了这一款 shoka 主题。这个主题不仅功能全，还漂亮，很多美图一下就击中了我这个老二次元的心（这才是主要原因吧喂）。
![nice](/home/cmaster/blog/source/_images/designer’s homepage.png)

​ 主题原作者小姐姐的成品页面，顺便给她打个广告

在作者主页上有详细的该主题安装、部署教程，这里我就略过了。由于本人对于相关的前端知识仅是一知半解，所以目前本博客还有许多功能待实现，以后我会写篇博客记录。

# 首次发博

谁想到第一次写博客还让我折腾了好一会儿呢。。。第一次上传文章内容竟离奇地不能显示，只好下载一个 typora markdown 编辑器来修改，总算是改好了。现在还有本地图片不能显示的问题存在。以及不能添加 tag date.

# 想要说的话

总之在经历了摸爬滚打之后，这个博客还是在农历新年之前上线了，同时也发布了这样一篇博文。尽管访问者可能有且仅有我自己，但我相信一切都会慢慢好起来的。赶在新年之前熬夜写好这篇博文，也是图个新年新气象。这下终于有个地方可以供我畅所欲言了！By the way, 如果有人对这个博客感兴趣的话可联系我 qq：1184215672 咱们互相添个友链引流什么的。
好啦，这么多内容，应该足够成为一篇还说得过去的 “开博词” 了吧！既然花这么多心思建了博客，就要多多在上面输出内容哦！（人在家中坐，flag 自天上来 orz）
最后，祝你新年快乐亚🎆！
